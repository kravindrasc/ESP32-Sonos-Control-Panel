# 3D Print Files

## ESP32 3.5 Display Frame.3mf

Picture-frame style desktop stand for the [3.5" ESP32 display board](https://www.amazon.com/dp/B0G3WDGSWG). Holds the board upright at a comfortable viewing angle with a cable exit at the back for the USB-C power cable.

**Compatible slicers:** PrusaSlicer, Bambu Studio, and any slicer that supports `.3mf`.
